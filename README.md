# CodeQuest Kids Edition v2.1 🎮

The complete, addictive browser game that teaches kids **Bash, Python, Java, JavaScript, and HTML+CSS**.

## 🚀 Deploy on Render.com (Recommended)

### Quick Deploy Steps

1. **Create a GitHub repo** (free)
2. Upload these files (or push with git):
   - `index.html`
   - `render.yaml`
3. Go to [Render.com](https://dashboard.render.com)
4. Click **+ New** → **Static Site**
5. Connect your GitHub repo
6. Deploy!

**Full step-by-step guide**: See **[README-render.md](README-render.md)**

## How to Play Locally

Just open `index.html` in any browser.

Or run a local server:
```bash
python3 -m http.server 8080
# Then visit http://localhost:8080
```
